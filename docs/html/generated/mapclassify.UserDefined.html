mapclassify.UserDefined¶

class mapclassify.UserDefined(y, bins)[source]¶

User Specified Binning

Parameters
yarray

(n,1), values to classify

binsarray

(k,1), upper bounds of classes (have to be monotically increasing)

Notes

If upper bound of user bins does not exceed max(y) we append an additional bin.

Examples

>>> import mapclassify as mc
>>> cal = mc.load_example()
>>> bins = [20, max(cal)]
>>> bins
[20, 4111.45]
>>> ud = mc.UserDefined(cal, bins)
>>> ud.bins
array([  20.  , 4111.45])
>>> ud.counts
array([37, 21])
>>> bins = [20, 30]
>>> ud = mc.UserDefined(cal, bins)
>>> ud.bins
array([  20.  ,   30.  , 4111.45])
>>> ud.counts
array([37,  4, 17])
